3 Methods

3.1 Isolation and Quantitative analysis of RNA

RNA were extracted from Plant Root samples using the PVP extraction method. This method is optimized for plant tissues containing high levels of polyphenols and polysaccharides. Following extraction, RNA quantity was measured using a Qubit® 4.0 fluorometer, and quality was analyzed by 1% agarose gel electrophoresis to ensure the absence of degradation.

3.2 Preparation of Library

The paired-end sequencing library was prepared using the KAPA mRNA HyperPrep Kit for Illumina (CAT #KK8581). The process was initiated with 500-1000 ng of input RNA. The workflow included mRNA enrichment, fragmentation, first and second-strand cDNA synthesis, end-repair, 3´ adenylation, and adapter ligation.

Selective enrichment of adapter-ligated DNA fragments was achieved through PCR amplification. The resulting libraries were validated for size distribution and concentration.

3.4 Cluster Generation and Sequencing

Library molecules bind to complementary adapter oligos on the paired-end flow cell. Cluster generation and sequencing were performed on the Illumina Novaseq X Plus platform. Paired-end sequencing allows template fragments to be sequenced from both directions, improving alignment accuracy and detecting structural variations.

4 Library Quality Control

Amplified libraries were analyzed on a TapeStation 4150 (Agilent Technologies) using High Sensitivity D1000 ScreenTape® to determine mean peak sizes and ensure proper fragment distribution.

Figure 4.1: Mean library peak size distribution (bp) validated via TapeStation 4150.

4.1 Wet Lab Inferences

The average sizes of the generated libraries were determined to be 479bp, 503bp, 487bp, 539bp, 344bp, 356bp, 367bp, and 353bp respectively for the 8 samples. All libraries met the concentration and size requirements for high-quality sequencing on the Novaseq X Plus platform.

5 Bioinformatics Analysis

5.1 Bioinformatics Workflow

Raw data was filtered to remove adapters and low-quality bases. Clean reads thus produced were used for analysis. Reference-guided transcript assembly was performed by mapping clean reads onto the reference genome using STAR (v2.7.10a) and assembling transcripts with StringTie (v2.2.1).

A consensus set of transcripts was obtained using the StringTie merge function. Transcript abundance was estimated for each individual sample, and read counts were used for differential expression analysis using the edgeR package. Gene Ontology and pathway analysis were performed using Blast2go cli and KEGG-KAAS respectively.
